# mimir

To install dependencies:

```bash
bun install
```

To run tests:

```bash
bun dev
```

To run the REPL:

```bash
bun run repl
```

REPL Options:

- `--tokens` or `-t`: Print tokens for each statement
- `--ast` or `-a`: Print AST for each statement
- `--help` or `-h`: Show help message

Example:

```bash
bun repl --tokens --ast
```

This project was created using `bun init` in bun v1.2.10. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
